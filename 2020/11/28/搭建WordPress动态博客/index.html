    云服务器自动搭建

    以腾讯云服务器为例：

    1. 点击【重置应用】

    2. 点击【应用镜像】，选择【WordPress】，勾选上确定声明后点击【确定】

    3. 点击【应用管理】

    4. 点击【登录】

    5. 打开 credentials.txt ，并记住其中的密码

    1. 登录管理员登录地址，然后输入刚才的账户名和密码

    2. 现在可以登录 WordPress 配置和撰写博客了
